## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- Browser
- Vs Code
### Setup
```sh
cd my-folder
  git clone git@github.com:belaymit/updated-portfolio.git
```
### Install
```sh
  cd updated-portfolio
  npm install
```
### Usage
```sh
    npm run dev
```
### Run tests
```sh
    npm run test
```
### Deployment
You can deploy this project:
- GitHub pages
